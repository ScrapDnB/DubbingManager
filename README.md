# Dubbing Manager

[![Python 3.8+](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/downloads/)
[![PySide6](https://img.shields.io/badge/PySide6-6.10.2-green.svg)](https://pypi.org/project/PySide6/)

**Dubbing Manager** — профессиональное приложение для управления проектами дубляжа и озвучивания видеоконтента.

![Интерфейс](docs/screenshot.png)

## ✨ Возможности

### 📋 Управление проектами
- Создание, сохранение и загрузка проектов в формате JSON
- Автосохранение каждые 5 минут
- Резервное копирование (функция "Копия")
- Drag & Drop для загрузки проектов и ASS файлов

### 👥 Актёрская база
- Централизованная база всех актёров проекта
- Назначение актёров на роли персонажей
- Цветовое кодирование для визуального различия
- Массовое назначение актёра на несколько персонажей
- Редактирование ролей актёра

### 🎬 Работа с эпизодами
- Управление сериями/эпизодами
- Импорт ASS файлов с авто-распознаванием тайм-кодов и персонажей
- Редактирование текста реплик
- Переименование персонажей с авто-обновлением во всех окнах
- Привязка видеофайлов к эпизодам
- Удаление эпизодов из проекта

### 🎥 Видео
- Встроенный видеоплеер
- Синхронизация: клик по реплике → переход к тайм-коду
- Отображение текущей реплики при воспроизведении

### 🔍 Поиск
- Глобальный поиск по всем эпизодам
- Фильтрация по актёрам
- Быстрая навигация к результатам

### 📊 Экспорт
| Формат | Возможности |
|--------|-------------|
| **HTML** | Таблица/сценарий, редактирование, подсветка актёров |
| **Excel** | Цветовая схема, настройки колонок |
| **Reaper RPP** | Регионы с маркерами, цвета дорожек |

### 🎙️ Телесуфлёр
- Автоматическая прокрутка текста
- Плавная прокрутка с настраиваемой задержкой
- Фокусировка на активной реплике
- Настройка цветов и размеров шрифтов
- Зеркальное отображение
- OSC синхронизация с Reaper

### 📈 Аналитика
- Сводный отчёт проекта (актёры, реплики, время)
- Статистика по эпизоду
- Распределение реплик по актёрам

## 🚀 Быстрый старт

### Установка

```bash
# Клонирование репозитория
git clone <repository-url>
cd DubbingManager

# Создание виртуального окружения
python3 -m venv .venv
source .venv/bin/activate  # macOS/Linux
# или
.venv\Scripts\activate     # Windows

# Установка зависимостей
pip install -r requirements.txt
```

### Запуск

```bash
python main.py
```

### Сборка исполняемого файла

```bash
# macOS
pyinstaller --noconsole --windowed --name "Dubbing Manager" main.py

# Windows
pyinstaller --noconsole --windowed --name "Dubbing Manager" main.spec
```

## 📖 Использование

### Создание проекта

1. Запустите приложение
2. Введите название проекта
3. Нажмите **Сохранить**

### Добавление актёра

1. Нажмите **+ Актер** в левой панели
2. Введите имя актёра
3. Выберите цвет из палитры
4. Назначьте роли через кнопку **Роли (N)**

### Импорт субтитров

1. Выберите серию из выпадающего списка
2. Нажмите **+ .ASS**
3. Выберите файл(ы) субтитров
4. Введите номер серии
5. Приложение автоматически распознает:
   - Тайм-коды (начало/конец)
   - Имена персонажей
   - Текст реплик

### Редактирование

| Действие | Как выполнить |
|----------|---------------|
| Изменить имя персонажа | Двойной клик по ячейке "Персонаж" |
| Назначить актёра | Выбрать из выпадающего списка в колонке "Актер" |
| Изменить текст | Двойной клик в таблице или через Live Preview |
| Сохранить ASS | Кнопка **💾 Сохранить ASS** |

### Экспорт

1. Нажмите **⚙ Настройки** для конфигурации экспорта
2. Выберите формат(ы): **Лист** (HTML) и/или **Excel**
3. Выберите **Текущая** серия или **Все**
4. Нажмите **ЭКСПОРТ**

### Телесуфлёр

1. Выберите серию
2. Нажмите **🎤 Телесуфлёр**
3. Настройте параметры:
   - Размеры шрифтов
   - Позицию линии чтения
   - Плавность прокрутки
   - Цветовую схему

## 📋 Форматы файлов

### Входные
| Расширение | Описание |
|------------|----------|
| `.ass` | Субтитры Advanced SubStation Alpha |
| `.mp4`, `.mkv`, `.mov` | Видеофайлы |
| `.json` | Проекты Dubbing Manager |

### Выходные
| Расширение | Описание |
|------------|----------|
| `.json` | Сохранение проекта |
| `.html` | Экспорт сценария (веб-формат) |
| `.xlsx` | Экспорт в Excel |
| `.rpp` | Проект REAPER (DAW) |

### Структура проекта (JSON)

```json
{
  "project_name": "Название проекта",
  "actors": {
    "actor_id": {
      "name": "Имя актёра",
      "color": "#RRGGBB"
    }
  },
  "global_map": {
    "Персонаж": "actor_id"
  },
  "episodes": {
    "1": "/path/to/episode1.ass"
  },
  "video_paths": {
    "1": "/path/to/video1.mp4"
  },
  "export_config": { ... },
  "prompter_config": { ... }
}
```

## 🐛 Решение проблем

### Приложение не запускается на macOS
```bash
# Проверьте права доступа
xattr -cr /Applications/Dubbing\ Manager.app
```

### Нет звука в видео
```bash
# Установите кодеки
brew install ffmpeg  # macOS
# или
pip install pyside6[ffmpeg]
```

### OSC не работает
1. Проверьте, что порты не заняты
2. Настройте брандмауэр
3. Включите "Local Network" в настройках приватности
