# Dubbing Manager

**DubbingManager** — это профессиональное приложение для управления проектами дубляжа (озвучивания) видеоконтента. Программа позволяет организовать работу над озвучиванием фильмов, сериалов и других видеоматериалов, управляя сценариями, актёрским составом, тайм-кодами и экспортом данных в различные форматы.

## Основные возможности

### 📋 Управление проектами
- **Создание и сохранение проектов** — полная поддержка создания новых проектов и сохранения их в собственный формат
- **Автосохранение** — автоматическое сохранение проекта каждые 5 минут
- **Загрузка существующих проектов** — работа с ранее сохранёнными проектами
- **Резервное копирование** — функция "Копия" для сохранения проекта под новым именем

### 👥 Управление актёрским составом
- **База актёров** — централизованное хранилище всех актёров проекта
- **Назначение ролей** — привязка актёров к персонажам/ролям
- **Цветовое кодирование** — присвоение уникальных цветов для визуального различия актёров
- **Удаление и редактирование** — полное управление описанными выше параметрами

### 🎬 Работа с эпизодами и сценариями
- **Управление эпизодами/сериями** — создание, переименование и организация эпизодов
- **Импорт ASS файлов** — импорт субтитров в формате ASS (Advanced SubStation Alpha) с автоматическим распознаванием времени начала/конца и имён персонажей
- **Редактирование текста** — встроенный редактор для изменения диалогов и реплик
- **Тайм-коды** — автоматическое управление временем начала и окончания каждой реплики
- **Привязка видео** — связывание каждого эпизода с видеофайлом

### 🎬 Воспроизведение видео
- **Встроенный видеоплеер** — просмотр видео прямо в приложении
- **Синхронизация с текстом** — при выборе строки в таблице видео перепрыгивает на нужный тайм-код
- **Просмотр реплик по видео** — таблица с текущей репликой при проигрывании видео

### 🔍 Поиск и фильтрация
- **Глобальный поиск** — поиск по всем репликам всех эпизодов проекта
- **Фильтрация по актёрам** — отображение реплик только выбранных актёров
- **Быстрая навигация** — быстрый переход к найденным репликам

### 📊 Экспорт данных
- **HTML экспорт** — создание красиво оформленных HTML документов с сценариями
- **Excel экспорт** — экспорт в XLSX формат с возможностью организации данных в таблицы
- **Массовый экспорт** — одновременный экспорт всех эпизодов
- **Настройки макета** — выбор между табличным макетом и другими форматами представления
- **Управление колонками** — выбор отображаемых колонок (ТК, персонаж, актёр, текст)
- **Размеры шрифтов** — настройка размеров шрифтов для каждого элемента
- **Применение цветов** — опция для подсветки реплик цветами актёров при экспорте
- **Выбор актёров для подсветки** — возможность выбрать, каких актёров подсвечивать в экспорте

### 🎙️ Суфлёр (Teleprompter) / Live Preview
- **Интерактивный суфлёр** — специальное окно для отображения текста во время записи озвучивания
- **Прокрутка текста** — автоматическая или ручная прокрутка текста
- **Зеркальное отображение** — опция для зеркального отражения текста
- **Косметические настройки**:
  - Размеры шрифтов для тайм-кода, персонажа, актёра и основного текста
  - Фокусировка на активной строке (регулировка коэффициента фокуса)
  - Цветовые схемы (фон, активный текст, неактивный текст, тайм-коды, имена актёров)
  - Показ/скрытие заголовков
- **HTML preview** — живой предпросмотр экспорта с возможностью редактирования текста
- **Редактирование на лету** — изменение текста прямо в режиме предпросмотра

### 🔗 Интеграция и синхронизация
- **OSC поддержка** — интеграция с другими приложениями через OSC protokol (Open Sound Control)
  - Входящий порт для получения данных о времени воспроизведения
  - Исходящий порт для отправки команд
  - Синхронизация с внешними приложениями
- **Reaper интеграция** — экспорт проектов в REAPER (DAW) с сохранением маркеров и цветов
  - Автоматическое преобразование цветов из HEX в формат Reaper
  - Создание проекта Reaper с региональными маркерами для каждой реплики

### 📈 Аналитика
- **Сводный отчёт проекта** — статистика по:
  - Общему количеству реплик
  - Количеству актёров
  - Общему времени озвучивания
  - Среднему времени реплики
  - Распределению реплик по актёрам

### 🛠️ Дополнительно
- **Горячие клавиши** — настраиваемые горячие клавиши для быстрой навигации в суфлёре (влево/вправо)
- **Drag & Drop** — перетаскивание файлов для быстрого импорта
- **Сохранение состояния** — запоминание последних использованных путей и настроек
- **Многоязычная поддержка** — интерфейс полностью на русском языке

## Требования к системе

### Зависимости
- **Python 3.8+**
- **PySide6** — фреймворк для создания графического интерфейса
- **python-osc** — поддержка OSC протокола
- **openpyxl** — работа с Excel файлами (опционально, для экспорта в XLSX)
- **requests** — HTTP клиент для веб-запросов

## Формат проекта

Проекты сохраняются в формате JSON и содержат:
```json
{
  "project_name": "Название проекта",
  "actors": {
    "actor_id": {
      "name": "Имя актёра",
      "color": "#RRGGBB",
      "roles": ["Роль 1", "Роль 2"]
    }
  },
  "episodes": {
    "ep_num": {
      "name": "Название эпизода",
      "lines": [
        {
          "id": "line_id",
          "in": "00:00:00.00",
          "out": "00:00:05.00",
          "character": "Имя персонажа",
          "actor_id": "actor_id",
          "text": "Текст реплики"
        }
      ]
    }
  },
  "video_paths": {
    "ep_num": "/path/to/video.mp4"
  },
  "export_config": { /* Настройки экспорта */ },
  "prompter_config": { /* Настройки суфлёра */ }
}
```

## Файловые форматы

### Входящие форматы
- **.ass** — SubStation Alpha субтитры (основной входящий формат)
- **.mp4**, **.mkv**, **.mov** — видеофайлы (поддерживаемые видео форматы)
- **.json** — проекты Dubbing Manager

### Выходящие форматы
- **.json** — сохранение проектов
- **.html** — экспорт сценариев в веб-формате
- **.xlsx** — экспорт в Excel
- **.ass** — сохранение отредактированных субтитров
- **.rpp** — проекты REAPER

## Интерфейс

### Главное окно
- **Левая панель** — база актёров с управлением ролями и цветами
- **Центральная часть** — работа с эпизодами, редактирование реплик, таблица сценария
- **Верхняя строка** — управление проектом (создание, открытие, сохранение)

### Специальные окна
- **Teleprompter** (Суфлёр) — окно для показа текста во время записи
- **HtmlLivePreview** — живой предпросмотр экспорта
- **GlobalSearch** — поиск по всему проекту
- **ProjectSummary** — статистика проекта

## Горячие клавиши

В режиме Teleprompter (настраиваемые):
- **Левая стрелка** — перейти к предыдущей реплике
- **Правая стрелка** — перейти к следующей реплике

## Описание основных операций

### Создание нового проекта
1. Запустите приложение
2. Введите название проекта в поле "Проект:"
3. Нажмите "Сохранить"

### Добавление актёра
1. Нажмите кнопку "+ Актер"
2. Введите имя актёра
3. Выберите цвет для актёра
4. Добавьте роли (персонажей), которых озвучивает актёр

### Импорт субтитров
1. Выберите эпизод из списка
2. Нажмите "+ .ASS"
3. Выберите файл в формате ASS
4. Программа автоматически распознает реплики, тайм-коды и имена персонажей

### Редактирование реплик
1. Выберите серию
2. В таблице дважды кликните на ячейку для редактирования
3. Измените необходимые поля (текст, персонаж, актёр, тайм-коды)
4. Изменения автоматически сохраняются

### Экспорт проекта
1. Выберите "Экспорт" в меню или на панели инструментов
2. Выберите формат (HTML, Excel)
3. Настройте параметры (макет, колонки, шрифты, цвета)
4. Нажмите "Экспортировать"
5. Выберите папку для сохранения

## Советы по использованию

- **Автосохранение** — проект автоматически сохраняется каждые 5 минут, но рекомендуется нажимать "Сохранить" для прямого сохранения
- **Резервное копирование** — используйте функцию "Копия" для создания резервных копий проекта
- **Горячие клавиши в Teleprompter** — отредактируйте горячие клавиши в "Настройки суфлёра" для удобства
- **Фильтрация по актёрам** — используйте фильтр в окне предпросмотра для показа реплик только нужных актёров
- **Live Preview** — проверяйте результаты экспорта в реальном времени перед финальным сохранением
