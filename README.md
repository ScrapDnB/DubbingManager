# Dubbing Manager

[![Python 3.8+](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/downloads/)
[![PySide6](https://img.shields.io/badge/PySide6-6.10.2-green.svg)](https://pypi.org/project/PySide6/)

**Dubbing Manager** — приложение для управления проектами дубляжа и озвучивания: импорт сценариев, распределение ролей, рабочие тексты, телесуфлёр, видеопросмотр, статистика и экспорт монтажных листов.

![Screenshot](Screenshot.png)

## ✨ Возможности

- **Управление проектами** — создание, сохранение, автосохранение, drag & drop
- **Актёрская база** — централизованная база актёров с цветовым кодированием, переименованием, удалением и Undo/Redo
- **Назначения ролей** — глобальные назначения на проект и локальные назначения только для одной серии
- **Папка проекта** — автоматический поиск и связывание файлов в папке и подпапках
- **Мониторинг файлов** — окно состояния файлов с цветовым кодированием и перепривязкой
- **Работа с эпизодами** — импорт ASS, SRT, DOCX файлов, создание рабочих текстов, редактирование реплик, привязка видео
- **Видеоплеер** — синхронизация с репликами, переход по тайм-кодам
- **Поиск** — глобальный поиск по всем рабочим текстам и фильтрация по актёрам
- **Экспорт** — HTML, Excel, Reaper RPP
- **Телесуфлёр** — автопрокрутка, OSC синхронизация с Reaper, переключение серий с сохранением настроек
- **Аналитика** — сводные отчёты, статистика проекта и статистика выбранного персонажа
- **Глобальные настройки** — настройки экспорта, телесуфлёра и объединения реплик сохраняются между проектами
- **Отмена действий** — Undo/Redo для всех основных операций (Ctrl+Z / Ctrl+Shift+Z)

## 🚀 Быстрый старт

```bash
# Клонирование и установка
git clone <repository-url>
cd DubbingManager
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# Запуск
python main.py
```

## 📖 Основы использования

1. **Создание проекта**: введите название и нажмите **Сохранить**
2. **Установка папки проекта**: нажмите **📁 Папка** и выберите папку с файлами
3. **Добавление актёра**: нажмите **+ Актер**, введите имя и выберите цвет
4. **Удаление актёра**: выберите актёра в таблице и нажмите **- Актер**
5. **Отмена действий**: используйте **Ctrl+Z** для отмены или **Ctrl+Shift+Z** для повтора
6. **Импорт сценария**: выберите серию, нажмите **📥 Импорт**, укажите ASS, SRT или DOCX
7. **Рабочий текст**: после импорта приложение создаёт редактируемый рабочий текст в папке `texts_dm`
8. **Настройка объединения реплик**: кнопка «🔗 Настройки объединения» — параметры применяются при создании рабочих текстов и в отчётах
9. **Назначение ролей**: выберите актёра в таблице. Колонка **Область** задаёт назначение глобально или только для текущей серии
10. **Экспорт**: настройте параметры и выберите формат (HTML/Excel/Reaper)

## Рабочие тексты

ASS, SRT и DOCX используются как источники импорта. После импорта Dubbing Manager создаёт рабочий текст серии в JSON-файле и дальше использует его для:

- монтажных листов;
- телесуфлёра;
- глобального поиска;
- статистики серии и проекта;
- статистики персонажа в правой панели.

Изменения текста сохраняются в рабочий JSON. Сохранение правок обратно в ASS/SRT отключено намеренно.

Если файлы проекта были перенесены, окно **📋 Файлы** помогает найти и перепривязать исходники, видео и рабочие тексты.

## Назначения ролей

По умолчанию роль назначается **глобально**: один и тот же персонаж получает одного актёра во всех сериях.

Для повторяющихся технических ролей вроде «Массовка», «Мужчина 1» или «Голос» можно переключить колонку **Область** в режим **Серия**. Тогда назначение актёра действует только в текущей серии и не меняет другие серии.

### Импорт DOCX

Приложение поддерживает импорт DOCX файлов с гибкой настройкой колонок:
- Нажмите **📥 Импорт** и выберите DOCX-файл
- Настройте маппинг колонок (имя персонажа, тайминг, текст)
- Проверьте предпросмотр и нажмите **Импортировать**
- Поддерживается тайминг в одной колонке через разделитель (например, `00:00:01,000 - 00:00:03,000`)
- Последние выбранные настройки колонок сохраняются для следующих импортов

Подробнее в [docs/DOCX_IMPORT.md](docs/DOCX_IMPORT.md)

## Телесуфлёр

Телесуфлёр использует рабочие тексты, поддерживает автопрокрутку и OSC-синхронизацию с Reaper. Внутри окна можно переключать серии без сброса выбранного актёра и настроек синхронизации.

## 🔧 Требования

- Python 3.8+
- macOS 10.15+, Windows 10+, Linux

## 🍎 macOS: Подпись приложения

На macOS приложение требует подписи кода для корректной работы (в частности, для доступа к событиям клавиатуры и другим системным функциям).
```bash
codesign --force --deep -s - /Applications/DubbingManager.app
```

|-----|------------|

| Видео | `.mp4`, `.mkv`, `.avi`, `.mov`, `.m4v`, `.wmv` |
| Проект и рабочие тексты | `.json` |
| Экспорт | `.html`, `.xlsx`, `.rpp` |
