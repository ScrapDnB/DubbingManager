# Окно «Файлы проекта»

## Обзор

Окно **«Файлы проекта»** предоставляет наглядное представление о всех файлах проекта с отображением их статуса (найден/не найден) и возможностью быстрой перепривязки.

## Открытие окна

Нажмите кнопку **📋 Файлы** в верхней панели окна приложения.

## Элементы окна

### Дерево файлов

Отображает структуру проекта в виде дерева:

```
📁 Серия 1
  📄 Субтитры (.ass)    ✓ Найден    /path/to/file.ass
  🎬 Видео              ✓ Найден    /path/to/video.mp4
📁 Серия 2
  📄 Субтитры (.ass)    ✗ Не найден /missing/file.ass
  🎬 Видео              ✗ Не найден /missing/video.mp4
```

### Статусы файлов

| Статус | Значение | Цвет |
|--------|----------|------|
| ✓ Найден | Файл существует по указанному пути | 🟢 Зелёный |
| ✗ Не найден | Файл отсутствует по указанному пути | 🔴 Красный |
| Не указан | Путь к файлу не задан | ⚪ Серый |

### Статистика

Внизу окна отображается статистика:
```
Всего файлов: 10 | ✓ Найдено: 8 | ✗ Не найдено: 2
```

## Возможности

### Перепривязка файла

Если файл не найден или перемещён:

1. Выберите файл в дереве (кликните по строке с файлом)
2. Нажмите кнопку **📁 Перепривязать...**
3. В диалоге выберите правильный файл
4. Путь автоматически обновится в проекте
5. Статус изменится на «✓ Найден»

### Обновление статусов

Для обновления статусов файлов (например, после перемещения файлов внешней программой):

1. Нажмите кнопку **🔄 Обновить**
2. Статусы всех файлов будут проверены заново

### Просмотр путей

Полные пути к файлам отображаются в последнем столбце. При наведении курсора на ячейку пути отображается всплывающая подсказка с полным путём.

## Примеры использования

### Сценарий 1: Проверка целостности проекта

Перед началом работы откройте окно «Файлы проекта» и убедитесь, что все файлы имеют статус «✓ Найден».

### Сценарий 2: Восстановление после перемещения

Если вы переместили папку с файлами:

1. Откройте проект
2. Укажите новую папку проекта (📁 Папка)
3. Откройте «Файлы проекта»
4. Для файлов со статусом «✗ Не найден» используйте перепривязку

### Сценарий 3: Поиск проблемных файлов

Отсортируйте файлы по статусу (визуально) для быстрого определения отсутствующих файлов.

## Сочетания клавиш

| Клавиша | Действие |
|---------|----------|
| `Enter` | Открыть перепривязку для выбранного файла |
| `F5` | Обновить статусы файлов |
| `Esc` | Закрыть окно |

## Технические детали

### Кэширование

Статусы файлов проверяются при открытии окна. Для принудительного обновления используйте кнопку «Обновить».

### Производительность

Для проектов с большим количеством файлов (>100) проверка статусов может занять несколько секунд.

### Интеграция с папкой проекта

Если указана папка проекта, окно автоматически проверяет файлы в этой папке при загрузке.
