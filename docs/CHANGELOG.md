# Changelog

## [1.7.0] - 2026-07-10

### Добавлено

- Новый модуль «Аудиосериал» для импорта книг из PDF, ручной разметки текста по персонажам и сохранения глав как серий проекта.
- Визуальный редактор границ глав: главы можно переименовывать, удалять, добавлять заново и вручную двигать границы по полному тексту книги.
- Быстрые слоты разметки персонажей: можно назначить персонажа и актёра в слот и применять роль к выделенному тексту горячей клавишей.
- Глобальные настройки аудиокниг с редактируемыми ключевыми словами для распознавания глав и разделов.
- Тип проекта «аудиокнига/аудиосериал»: главы сохраняют порядок книги, а subtitle-проекты продолжают сортировать серии по номеру.
- Плавающий пульт телесуфлёра теперь умеет выбирать серию и показывает текущую реплику прямо в списке пульта.
- На Windows добавлен скрипт регистрации `.dub`-проектов для открытия двойным кликом.

### Изменено

- Импорт PDF лучше сохраняет книжную структуру текста, заголовки глав и оформление при разметке.
- Главы из PDF передаются в основную таблицу проекта вместе с персонажами и назначениями актёров из актёрской базы.
- Телесуфлёр корректно работает с импортированными рабочими текстами и позволяет переключать серии из основного окна и пульта.
- Окно аудиосериала открывается выше, чтобы боковая панель помещалась целиком.
- Экспорт в Reaper получил больше данных по проекту и удобнее переносит структуру для записи.

## [1.6.1] - 2026-06-14

### Добавлено

- Экспорт монтажного листа в PDF для одиночной серии, пакетного экспорта и быстрого конвертера ASS/SRT.
- Массовое назначение ролей одному актёру из правой панели: роли выбираются галочками, затем назначаются выбранному актёру.
- Экспорт XLSX из сводного отчёта проекта в формате, удобном для Google Sheets: персонажи, актёры, серии, сумма выбранной метрики и вертикальная разбивка новых персонажей по сериям.
- При экспорте сводного отчёта можно выбрать кольца, строчки или слова; выбор сохраняется глобально.
- Форматированный XLSX сводного отчёта подсвечивает шапку и разделители серий жёлтым, включает перенос по словам и расширяет колонки персонажа и актёра.

## [1.6.0] - 2026-06-12

### Главное

- Проведён крупный рефакторинг главного окна: UI-сборка, модель таблицы, импорт, Reaper-экспорт, глобальные актёры, настройки и quick-preview вынесены в отдельные модули и контроллеры.
- Быстрое преобразование subtitle-файлов в монтажный лист стало отдельным сценарием с drag-and-drop зоной и изолированной логикой обработки.
- Экспорт в Reaper RPP вынесен в отдельный сервис с предпросмотром, поддержкой видео, регионов и транслитерации имён дорожек.

### Добавлено

- Сервис статистики персонажей для переиспользования данных по ролям и сериям.
- Сервис навигационной логики телесуфлёра и отдельные виджеты телесуфлёра.
- Отдельные helper-модули для настроек, предпросмотра монтажного листа и совместимости проектных файлов.
- Тесты для новых контроллеров, helper-модулей, Reaper RPP, логики объединения реплик, телесуфлёра и миграций проекта.

### Изменено

- `ExportService` стал фасадом: объединение реплик и генерация Reaper RPP вынесены в специализированные сервисы.
- `ProjectService` больше не хранит правила legacy-миграций внутри основного класса.
- `ScriptTextService` получил отдельные методы для проверки и связывания рабочих текстов серий.
- Главное окно стало меньше отвечать за бизнес-логику и больше работать как координатор UI.

### Исправлено

- Быстрый предпросмотр subtitle-файлов больше не наследует актёрские цвета и фильтры проекта.
- Совместимость старых проектов покрыта отдельными тестами после выноса миграционной логики.

## [1.4.4] - 2026-06-07

### Главное

- В окно «О программе» добавлена проверка обновлений через GitHub Releases.
- Добавлен прототип автоматического обновления: исходники обновляются через `git pull --ff-only`, а собранная версия скачивает релизный артефакт и запускает внешний updater.

### Добавлено

- `UpdateService` для проверки последнего GitHub Release, сравнения версий, выбора релизного артефакта под macOS/Windows и скачивания обновления.
- Source-update для запуска из Python-исходников: обновление выполняется только из чистого git checkout.
- Binary-update для PyInstaller-сборок:
  - macOS скачивает `Dubbing_Manager_macOS.dmg`, закрывает приложение, заменяет текущий `.app` и запускает новую версию;
  - Windows скачивает `Dubbing_Manager_Windows.zip`, закрывает приложение, заменяет папку приложения и запускает `.exe`.
- Fallback на страницу релиза, если автообновление недоступно, платформа не поддержана или релизный asset не найден.

### Изменено

- Кнопка проверки обновлений размещена в окне «О программе», рядом с версией приложения и ссылкой на GitHub.
- Проверка обновлений теперь предлагает автоматическую установку, а при отказе открывает страницу релиза.

### Сборка

- Статистика по персонажу.
- Новые настройки импорта DOCX и сохранение этих настроек.
