# Импорт DOCX файлов

## Обзор

Приложение поддерживает импорт файлов DOCX с гибкой настройкой колонок. Вы можете указать, в каких колонках таблицы находится информация для каждого поля.

## Формат файла

Файл DOCX должен содержать таблицу со следующими возможными колонками:

| Поле | Описание | Пример |
|------|----------|--------|
| **Имя персонажа** | Кто произносит реплику | `Иван`, `Мария` |
| **Тайминг (начало)** | Время начала реплики (отдельная колонка) | `00:00:01,000` |
| **Тайминг (конец)** | Время конца реплики (отдельная колонка) | `00:00:03,000` |
| **Тайминг (вместе)** | Начало и конец в одной колонке через разделитель | `00:00:01,000 - 00:00:03,000` |
| **Текст фразы** | Текст реплики | `Привет! Как дела?` |

**Примечание:** Используйте либо раздельные колонки "Тайминг (начало)" и "Тайминг (конец)", либо одну колонку "Тайминг (вместе)".

## Поддерживаемые форматы времени

- `ЧЧ:ММ:СС,мсс` (например, `00:01:30,500`)
- `ЧЧ:ММ:СС.мсс` (например, `00:01:30.500`)
- `ММ:СС,мсс` (например, `01:30,500`)
- `ММ:СС` (например, `01:30`)

## Разделители для тайминга в одной колонке

По умолчанию поддерживаются следующие разделители:
- `-` (дефис)
- `–` (среднее тире, en-dash)
- `—` (длинное тире, em-dash)
- `|` (вертикальная черта)
- `/` (слэш)

Вы можете настроить разделители в диалоге импорта в поле "Разделители тайминга".

## Использование

### 1. Нажмите кнопку "+ .DOCX"

В панели управления сериями нажмите кнопку **"+ .DOCX"**.

### 2. Выберите файл

В диалоге выбора файла укажите путь к вашему DOCX файлу.

### 3. Настройте маппинг колонок

После загрузки файла откроется окно настройки, где вы сможете:

- **Автоопределение**: нажмите "🔍 Автоопределение" для автоматического распознавания колонок по заголовкам
- **Ручная настройка**: для каждого поля выберите нужную колонку из выпадающего списка
- **Настройка разделителей**: укажите разделители для тайминга в одной колонке (например, `- | /`)

### 4. Проверьте предпросмотр

В правой части окна отображается таблица предпросмотра:
- Зелёные галочки (✓) — корректные строки
- Жёлтые предупреждения (⚠) — строки с проблемами (например, нет текста)

### 5. Импортируйте данные

Нажмите **"✅ Импортировать"** для импорта данных в проект.

## Примеры заголовков для автоопределения

Система автоматически распознаёт следующие заголовки:

| Поле | Распознаваемые заголовки |
|------|-------------------------|
| Имя персонажа | `персонаж`, `имя`, `actor`, `character`, `char`, `voice` |
| Тайминг (начало) | `начало`, `start`, `time start`, `in`, `from` |
| Тайминг (конец) | `конец`, `end`, `time end`, `out`, `to` |
| Тайминг (вместе) | `тайминг`, `timing`, `time`, `время` |
| Текст фразы | `текст`, `text`, `replica`, `фраз`, `dialog`, `speech` |

## Примеры таблиц

### Вариант 1: Раздельные колонки тайминга

```
┌──────────┬──────────────┬────────────┬──────────────────┐
│ Персонаж │ Начало       │ Конец      │ Текст            │
├──────────┼──────────────┼────────────┼──────────────────┤
│ Иван     │ 00:00:01,000 │ 00:00:03,0 │ Привет!          │
│ Мария    │ 00:00:04,000 │ 00:00:06,0 │ Как дела?        │
│ Иван     │ 00:00:07,000 │ 00:00:09,0 │ Нормально, спасибо│
└──────────┴──────────────┴────────────┴──────────────────┘
```

### Вариант 2: Тайминг в одной колонке

```
┌──────────┬─────────────────────────────┬──────────────────┐
│ Персонаж │ Тайминг                     │ Текст            │
├──────────┼─────────────────────────────┼──────────────────┤
│ Иван     │ 00:00:01,000 - 00:00:03,000 │ Привет!          │
│ Мария    │ 00:00:04,000 - 00:00:06,000 │ Как дела?        │
│ Иван     │ 00:00:07,000 - 00:00:09,000 │ Нормально, спасибо│
└──────────┴─────────────────────────────┴──────────────────┘
```

## Отмена импорта

Если вы импортировали данные с ошибкой, используйте **Ctrl+Z** для отмены действия.

## Требования

- Python пакет `python-docx` (устанавливается автоматически через `requirements.txt`)
- Файл DOCX должен содержать таблицу в стандартном формате Word
