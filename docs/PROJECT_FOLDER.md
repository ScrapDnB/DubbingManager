# Папка проекта (Project Folder)

## Обзор

Функция **"Папка проекта"** позволяет привязать к проекту корневую папку, в которой хранятся все связанные файлы: субтитры (ASS) и видеофайлы. Программа автоматически находит файлы в папке и подпапках, даже если структура папок отличается от ожидаемой.

## Возможности

- **Автоматический поиск файлов** — рекурсивное сканирование папки и всех подпапок
- **Умное распознавание серий** — поддержка различных форматов именования файлов
- **Автоматическое связывание** — найденные файлы автоматически добавляются в проект
- **Обновление путей** — если файлы были перемещены, программа найдёт новые пути
- **Undo/Redo** — установка и очистка папки проекта поддерживают отмену действий

## Использование

### Установка папки проекта:

1. Нажмите кнопку **📁 Папка** в верхней панели
2. Выберите корневую папку вашего проекта
3. Программа автоматически просканирует папку и найдёт файлы
4. Найденные файлы будут добавлены в проект

### Отвязка папки проекта:

1. Нажмите кнопку **🔓** рядом с названием папки
2. Подтвердите отвязку

> ⚠️ **Внимание:** При отвязке папки файлы остаются в проекте, но автоматический поиск прекращается.

### Повторное сканирование:

При загрузке проекта с указанной папкой программа автоматически:
- Проверяет наличие файлов по сохранённым путям
- Если файлы не найдены — ищет их в папке проекта
- Обновляет пути к файлам при необходимости

## Поддерживаемые форматы именования

Программа распознаёт номера серий в следующих форматах:

| Формат | Пример | Распознаётся как |
|--------|--------|------------------|
| S01E01 | `S01E01.mkv` | Серия 1, Эпизод 1 |
| S1E1 | `S1E1.ass` | Серия 1, Эпизод 1 |
| EP01 | `EP01.mkv` | Серия 1 |
| Ep01 | `Ep01.ass` | Серия 1 |
| Episode 01 | `Episode 01.mkv` | Серия 1 |
| - 01 | `Series - 01.ass` | Серия 1 |
| [01] | `[Subs] Series [01].mkv` | Серия 1 |
| 01 | `01.ass` | Серия 1 |
| 1 | `1.mkv` | Серия 1 |

### Для двухсерийных форматов (S01E01):
Формат `S01E01` преобразуется в `"1 1"` (серия 1, эпизод 1).

## Поддерживаемые форматы файлов

### Субтитры:
- `.ass`

### Видео:
- `.mp4`
- `.mkv`
- `.avi`
- `.mov`
- `.m4v`
- `.wmv`

## Структура папок

Программа поддерживает любую структуру папок. Например:

```
Project/
├── Subtitles/
│   ├── Series_E01.ass
│   ├── Series_E02.ass
│   └── Series_E03.ass
├── Video/
│   ├── Series_E01.mkv
│   ├── Series_E02.mkv
│   └── Series_E03.mkv
└── project.json
```

Или:

```
Project/
├── S01/
│   ├── 01.ass
│   └── 01.mkv
├── S02/
│   ├── 01.ass
│   └── 01.mkv
└── project.json
```

Обе структуры будут корректно обработаны.

## Особенности

### Игнорируемые файлы и папки:
- Скрытые файлы (начинаются с `.`)
- Скрытые папки (начинаются с `.`)
- Системные файлы

### Кэширование:
Для ускорения работы результаты сканирования кэшируются. Кэш автоматически обновляется при изменении папки проекта.

### Обновление путей:
Если файл был перемещён в пределах папки проекта, программа автоматически обновит путь при следующем сканировании.

## Просмотр структуры файлов

### Окно "Файлы проекта"

Для просмотра структуры файлов проекта нажмите кнопку **📋 Файлы** в верхней панели.

Окно отображает:
- **Дерево файлов** — эпизоды и связанные с ними файлы (ASS, видео)
- **Статус файлов** — ✓ Найден (зелёный) или ✗ Не найден (красный)
- **Пути к файлам** — полные пути к каждому файлу
- **Статистику** — общее количество, найдено, не найдено

### Возможности окна:

1. **Перепривязка файла**:
   - Выберите файл в дереве
   - Нажмите **📁 Перепривязать...**
   - Выберите правильный файл
   - Путь автоматически обновится в проекте

2. **Обновление**:
   - Нажмите **🔄 Обновить** для обновления статусов файлов

3. **Цветовое кодирование**:
   - ✓ **Зелёный** — файл найден
   - ✗ **Красный** — файл не найден
   - ⚪ **Серый** — путь не указан

## API

### ProjectFolderService

Основной сервис для управления папкой проекта:

```python
from services import ProjectFolderService

service = ProjectFolderService()

# Установка папки
service.set_project_folder(data, "/path/to/folder")

# Получение папки
folder = service.get_project_folder(data)

# Очистка
service.clear_project_folder(data)

# Сканирование и связывание
ass_count, video_count = service.scan_and_link_files(data, folder)

# Поиск всех файлов
found = service.find_all_media_files(folder)
# {"ass": {"1": "/path/1.ass"}, "video": {"1": "/path/1.mkv"}}

# Статистика
stats = service.get_folder_stats(folder)
# {"ass_count": 3, "video_count": 3, "episodes": ["1", "2", "3"]}
```

### SetProjectFolderCommand

Команда для Undo/Redo:

```python
from core.commands import SetProjectFolderCommand

# Установка папки
command = SetProjectFolderCommand(data, "/path/to/folder")
undo_stack.push(command)

# Очистка папки
command = SetProjectFolderCommand(data, None)
undo_stack.push(command)
```

## Примеры использования

### Сценарий 1: Новый проект
1. Создайте новый проект
2. Укажите папку проекта
3. Все файлы будут автоматически добавлены

### Сценарий 2: Перемещение файлов
1. Переместите файлы в новую папку
2. Откройте проект
3. Укажите новую папку проекта
4. Программа найдёт и обновит пути

### Сценарий 3: Работа с подпапками
1. Организуйте файлы в подпапках (Subtitles/, Video/)
2. Укажите корневую папку проекта
3. Программа найдёт файлы во всех подпапках

## Интеграция с проектом

Папка проекта сохраняется в файле проекта (`.json`) в поле `project_folder`. При загрузке проекта:
1. Проверяется наличие `project_folder`
2. Если папка указана — выполняется сканирование
3. Найденные файлы связываются с эпизодами
4. Отсутствующие файлы помечаются для поиска
